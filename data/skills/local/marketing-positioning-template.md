# Marketing Positioning Template

## Rationale
Pattern appeared 3+ times (surrogate, Vanguard, arkship) with identical sections — standardize to avoid generic 'X which Y' placeholder copy and enforce measurable-outcome framing.

## Steps
1. Write 2-sentence elevator pitch emphasizing measurable outcomes over features
2. Define Target ICP: company size, industry, role
3. List top 3 value props tied to concrete user outcomes
4. Write 1-paragraph competitor positioning with specific named competitors (no placeholder X/Y)
5. Draft landing page hero: headline + subhead + 3 benefit bullets + 3 objection handlers
