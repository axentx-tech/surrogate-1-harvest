# Growth Loop Experiments

## Rationale
surrogate and Costinel growth docs use identical loop structure (Acquisition/Activation/Retention × 2 ideas each) with hypothesis+metric+effort+timeframe+lift. Template makes experiments comparable and prioritizable.

## Steps
1. For each loop (acquisition, activation, retention) generate 2 experiment ideas
2. State hypothesis as a testable causal claim
3. Define one primary metric per experiment
4. Tag effort (S/M/L) and timeframe in weeks
5. Estimate expected lift % based on benchmarks or prior results
