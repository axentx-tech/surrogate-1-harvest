# Ai Ml Design

## Rationale
AI/ML enhancement design appeared 3 times with consistent template (Technique → Data → Model → Integration → Effort); standardizing prevents ad-hoc ML planning.

## Steps
1. Identify the ML technique category: supervised/unsupervised/anomaly detection/NLP/etc
2. Document data requirements: sources, features, labels if needed
3. Select model(s) with justification (e.g., Isolation Forest for high-dim anomaly)
4. Define integration point: APIs, pipelines, dashboards
5. Estimate effort by phase: data prep, training, integration, deployment
