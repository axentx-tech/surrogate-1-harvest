# Adr Write

## Rationale
ADR pattern appeared 4 times across 3 projects with consistent structure; standardizing ensures uniform quality and reduces cognitive load for architects.

## Steps
1. Gather context: current state, constraints, recent decisions, stakeholder needs
2. Identify the decision to be made and 2-3 alternatives considered
3. Write Decision section with concrete implementation approach
4. Document Consequences: positive, negative, and mitigations
5. Review against existing ADRs for consistency and cross-references
