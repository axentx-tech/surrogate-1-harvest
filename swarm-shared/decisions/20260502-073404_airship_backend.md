The highest-value incremental improvement that can ship in <2h is to optimize the Surrogate AI service's data ingestion pipeline by utilizing the HF CDN bypass pattern. This involves downloading public dataset files directly from the CDN without using the HF API, which is rate-limited.

### Implementation Plan
1. **Update the data ingestion script**: Modify the script to download dataset files from the HF CDN using the `https://huggingface.co/datasets/{repo}/resolve/main/{path}` URL pattern.
2. **Pre-list file paths**: Make a single API call to `list_repo_tree(path, recursive=False)` to retrieve the list of file paths for a specific date folder.
3. **Embed file list in training script**: Save the list of file paths to a JSON file and embed it in the training script (`train.py`) to enable CDN-only fetches during data loading.
4. **Test and verify**: Test the updated data ingestion pipeline to ensure it is working correctly and efficiently.

### Code Snippets
```python
import requests
import json

# Pre-list file paths
repo = "your-repo-name"
path = "your-path"
response = requests.get(f"https://huggingface.co/datasets/{repo}/tree/main/{path}")
file_paths = response.json()["files"]

# Save file list to JSON
with open("file_paths.json", "w") as f:
    json.dump(file_paths, f)

# Embed file list in training script
with open("file_paths.json", "r") as f:
    file_paths = json.load(f)

# Download dataset files from HF CDN
for file_path in file_paths:
    url = f"https://huggingface.co/datasets/{repo}/resolve/main/{file_path}"
    response = requests.get(url)
    with open(file_path, "wb") as f:
        f.write(response.content)
```
